les languages métrisants

PHP

PHP est un langage de programmation côté serveur utilisé pour créer des sites web dynamiques. Il permet de traiter des données, interagir avec des bases de données et générer du contenu HTML automatiquement.

JAVA-SCRIPT

JavaScript est un langage de programmation du web qui s’exécute principalement dans le navigateur. Il permet de rendre les pages web interactives et dynamiques (clics, animations, formulaires, etc.).

LARAVEL

Angular est un framework JavaScript/TypeScript développé par Google pour créer des applications web dynamiques et interactives.

ANGULAR

Laravel est un framework PHP moderne qui sert à créer des applications web (sites, dashboards, APIs…). Il est très populaire parce qu’il est clair, structuré et rapide à développer.

PYTHON

Python est un langage de programmation polyvalent, simple et lisible. Il est utilisé pour le développement web, la science des données, l’intelligence artificielle, l’automatisation, et bien plus.

FLUTTER

Flutter est un framework de Google utilisé pour créer des applications mobiles, web et desktop à partir d’un seul code. Il utilise le langage Dart et permet de développer des interfaces rapides et modernes.
